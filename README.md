## 先确认：你的 IP 真的被封了吗？

别还没确认就急着去换。搬瓦工 VPS 连不上有好几种可能——IP 被封是一种，但流量跑完被限速、VPS 非正常关机、本地网络问题，也会出现类似症状。

**官方检测方法：**

1. 登录 KiwiVM 控制面板（找到你的 VPS，点进去）
2. 面板内有 IP 黑名单检测工具，点击"Test Main IP"
3. 以这个结果为准，它从多个大陆节点做测试

如果检测结果显示国内大面积不通，基本就是被封了。

顺便提一嘴：IP 被封之后，在 KiwiVM 里选"切换机房"也没用——系统会检测到 IP 状态，直接报错拒绝迁移。免费切机房换 IP 这条路，封了之后走不通。
